Developer Reference / @visdesignlab/upset2-core / getQueryResult

Function: getQueryResult()

getQueryResult(rows, membership): Rows

Defined in: render.ts:116

Filters and returns rows based on the specified set membership query.

Parameters​

rows​

Rows

The rows to be filtered.

membership​

SetQueryMembership

An object representing the set membership query. The keys are set names and the values are 'Yes' or 'No' indicating whether the row should belong to the set or not.

Returns​

Rows

The filtered rows that match the set membership query.
