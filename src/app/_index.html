New version of Upset is available at upset.multinet.app

Dear users,

We are excited to announce the brand new version for Upset2.

The new version comes with an improved interface and user experience. Some the highlights of the new version are:

  • Improved dataset management leveraging the Multinet data upload interface.
  • Provenance support to undo/redo and load/save current Upset state.
  • Upset is now completely rewritten in React + Typescript.
  • We are coming up with new features like accessibilty, standalone upset component, and more in the near future.

Checkout the new version of UpSet at upset.multinet.app. Get started by uploading your dataset or exploring some public datasets here.

This website will be available till March 31st, 2023. After that this URL will redirect to the new version.

Make sure you reupload any dataset you were working with on this version and upload it to the new version. Datasets here will be deleted on April 1st, 2023 and won't be accessible any more.
